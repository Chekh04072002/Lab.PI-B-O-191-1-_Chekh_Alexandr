    2. Как изменить цветовую схему (оформление) среды?

Цветовая схема Visual Studio меняется через "Параметры" в графе "Средства" (по-старому "Сервис")

* Открыть Средства -> Параметры...
* Убедиться, что в левой части открывшегося окна выбрана вкладка "Окружение" с подпунктом "Общие"
* В правой части выбрать нужную цветовую тему

    3. Как закомментировать/раскомментировать блок кода средствами VS?

Комментирование и раскомментирование в Visual Studio производится двумя сочетаниями клавиш, предварительно поставив указатель мышки на нужную строчку или выделив нужную область.

* *Ctrl + K, C* комментирует строчку
* *Ctrl + K, U* убирает комментарий

    4. Как открыть в проводнике Windows папку с проектом средствами VS;

Открыть проводник для выбора папки можно двумя способами:

* Использовать сочетание клавиш *Ctrl + Shift + Alt + O*
* Перейти по Файл -> Открыть -> Папка...

После чего в проводнике нужно выбрать требуемую папку и нажать "Выбор папки"

    8. Как установить/убрать точку остановки (breakpoint)?

Чтобы установить точку останова, вам нужно кликнуть левой кнопкой мыши на серую вертикальную линию, находящуюся возле номеров строк (видно на скриншоте).
Чтобы её убрать, просто щёлкните правой кнопкой мыши на точку останова, которую вам нужно убрать, и нажмите"удалить точку останова".

![Изображение#15](https://sun9-69.userapi.com/c855528/v855528938/1acb24/fKVrWhIY9b8.jpg "Рис. 15")\

return 0;
